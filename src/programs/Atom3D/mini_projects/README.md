Atom3D Mini-Projects
=====================

This folder contains small, focused mini-projects designed to teach the core areas needed to port and extend `atom_realtime.cpp` into Python. Complete each mini-project to progressively gain confidence in the math, sampling, rendering, and performance techniques used by the Atom simulator.

Mini-projects (ordered):
- `01-array_math`
- `02-sampling_specialfuncs`
- `03-spherical_transforms`
- `04-rendering_basics`
- `05-rendering_instancing`
- `06-glsl_shaders`
- `07-performance_numba_gpu`
- `08-camera_ui_integration`
- `09-integration_harness`

Open each project's folder and follow its `README.md` for goals, core concepts, step-by-step plan, and verification parameters.
