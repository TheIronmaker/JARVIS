
## Atom Simulator Program
One of the core visual modules of this JARVIS instance is an interactive **Atom Simulator**. 

- **Language:** Python (utilizing PyOpenGL)
- **Origin:** This feature is a Python translation of the **Atoms** project originally written in C++ by [Kavan](https://x.com/KavanSG).
- **Inspiration:** You can view the original project breakdown on YouTube: [Creating an Atom Simulator](https://www.youtube.com). Or view his code on [GitHub](https://github.com/kavan010/Atoms)

*Special thanks to Kavan for open-sharing his development process and providing the C++ implementation of complex equations*
